
# Pomodoro Timer made with React and Material UI

[![pas](https://img.shields.io/static/v1?&message=ProgressiveApp.Store&color=74b9ff&style=flat&label=Follow%20Pomodoro%20%20at)](https://progressiveapp.store/pwa/Pomodoro-) [![Netlify Status](https://api.netlify.com/api/v1/badges/d31a3e8d-dd45-437c-99bf-6fe883c65a69/deploy-status)](https://app.netlify.com/sites/pomodoro-rgao/deploys)

A simple Pomodoro App that uses the Pomodoro timer to efficiently manage study time.

  

## Features

The session and break timers are configurable to your liking.

  

The default timer values are **25 minutes** per session and **5 minutes** per break.

**Note: Changing the session/break timers will pause the main timer.**

  

Clicking the reset button resets the timers to their default values.

  

## Setup

  

Clone the repository to your local machine.

  

Run `npm install` to install the necessary dependencies

  

Start the project with `npm start`
